# SnippetBox

SnippetBox is a simple, lightweight, and extensible code snippet manager for VS Code.

## Features

- SnippetBox is a simple, lightweight, and extensible code snippet manager for VS Code.
- SnippetBox is designed to be as simple as possible, while still providing a rich set of features.
- SnippetBox is built on top of the [Snippet API](https://code.visualstudio.com/api/language-extensions/snippet-guide), which allows for easy integration with other VS Code extensions.
- SnippetBox is designed to be extensible, allowing for easy customization and integration with other extensions.
